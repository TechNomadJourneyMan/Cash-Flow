# Digital Central Bank & Global Exchange

Неофициальный цифровой игровой модуль для финансовых настольных игр в стиле CashFlow:
цифровой Центробанк, глобальная биржа, симулятор экономических событий, финансовые
бланки игроков и единая realtime-экономика для большой офлайн-игры на несколько столов.

## Страницы

| Путь | Что это |
|---|---|
| `/` (→ `/firebase-portal.html`) | **Главное приложение** — realtime-портал на Firebase: роли админ / банкир / аудитор / игрок, кошельки, переводы, кредиты, биржа, пулы, венчур, события, бланки игроков |
| `/index.html` | Калькулятор событий и игроков (офлайн, localStorage) |
| `/digital-central-bank.html` | Single-device прототип портала (офлайн, localStorage) |

## Технологии

- Статический сайт (HTML/CSS/JS, без сборки).
- Firebase **Firestore** (realtime) + **Anonymous Auth** — серверная синхронизация между устройствами.
- Деплой на **Vercel** (статический хостинг).

## Настройка Firebase

Конфиг проекта уже вшит в `firebase-portal.html`. Для работы нужно:

1. В Firebase Console включить **Authentication → Anonymous**.
2. Создать **Firestore Database** и опубликовать правила из `firestore.rules`.
3. (После деплоя) добавить домен Vercel в **Authentication → Settings → Authorized domains**.

Подробности — в [`FIREBASE-SETUP.md`](FIREBASE-SETUP.md).

## Деплой на Vercel

Проект подключается к Vercel как обычный статический репозиторий — никаких build-команд не требуется.
`vercel.json` перенаправляет корень `/` на портал.

> Примечание: `apiKey` Firebase в веб-приложении не является секретом (это идентификатор проекта).
> Доступ к данным ограничивают правила Firestore.
